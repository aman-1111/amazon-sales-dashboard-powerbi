# Amazon Laptop Sales Dashboard (Power BI)

## Overview
This project features an interactive Power BI dashboard built to analyze Amazon laptop sales data. The dashboard focuses on sales performance, pricing trends, customer ratings, and operating system distribution to derive meaningful business insights.

The goal of this project was not just to create visuals, but to understand how to select the right metrics and charts to tell a clear business story.

---

## Key Insights
- Total sales and total units sold across laptop brands
- Average laptop price and average customer rating
- Brand-wise sales performance comparison
- Price vs rating relationship analysis
- Operating systemâ€“based sales distribution
- Product-level sales and stock details

---

## Dashboard Features
- KPI cards for quick performance overview
- Bar and column charts for brand comparison
- Scatter chart for price vs rating analysis
- Donut chart for OS distribution
- Interactive slicers for Brand, OS, RAM, and Price range
- Detailed table for drill-down analysis

---

## Tools & Technologies
- Power BI Desktop
- Power Query (data cleaning and transformation)
- CSV dataset

---

## Dataset
The dataset includes the following attributes:
- Brand and model
- Price
- Customer rating
- Operating system
- RAM configuration
- Sales count and available stock

---

## How to Use
1. Download the `.pbix` file
2. Open it using Power BI Desktop
3. Use slicers to explore insights by brand, OS, RAM, and price range
